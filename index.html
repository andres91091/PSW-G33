Introducción


Esta página se desarrolla durante la realización del curso Programación De Sitios Web el cual pertenece al pénsum del programa de Ingeniería De Sistemas de la UNAD.

Se desarrolla utilizando HTML como lenguaje de estructura, framework Bootstrap y CSS como lenguaje de estilo en su parte Front-End o del lado del cliente, y se utilizo JavaScript como lenguaje de programación permitiendo la conexión con la base de datos, se utiliza MySQL como motor de base de datos, esto en su parte Back-End o del lado del servidor.

En la pestaña de administrador se encuentra accesos para la creación de la base de datos, además de la creación de la tabla que usaremos, así como la generación de un PDF con la información alojada en la base de datos y un Backup para respaldar dicha información.

En la pestaña inventarios encontraremos 4 pestañas que nos llevaran a 4 diferentes formularios donde podremos ingresar la información del producto, actualizarla, eliminarla y consultarla.

Además contamos con la pestaña utilidades, en la cual tendremos dos formularios que nos permitirán realizar algunas operaciones para conocer la ganancia e impuestos de un producto, además de poder realizar la conversión de algunas medidas.
